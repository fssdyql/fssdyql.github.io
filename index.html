插件列表管理器 v1.4.7

基于Web浏览器的本地化插件管理工具，专为音乐插件（LX、Music Free、澜音等）开发者与使用者设计。无需安装，单HTML文件运行，数据完全存储在本地浏览器中。

纯前端运行 多标签页 自定义主题 数据备份 响应式设计

核心功能：

  • 标签页管理系统（内置+自定义）
  • 完整的插件项目管理（增删改查+排序+搜索）
  • 主题定制系统（预设主题+自定义模板）
  • 多种导出格式（HTML、Markdown、JSON）
  • 键盘快捷键与自动保存

技术特点

  • 纯前端技术栈：HTML5 + CSS3 + JavaScript
  • 数据存储：LocalStorage（支持数据压缩）
  • 文件处理：FileReader API + Base64编码
  • 拖拽排序：原生HTML5 Drag & Drop API
  • 导出功能：Blob API + 动态下载
  • 响应式设计：CSS Grid + Flexbox

使用场景：

  • 插件开发者：管理多个版本和变体
  • 插件使用者：整理收藏的插件列表
  • 团队协作：通过导出HTML分享插件集合
  • 文档编写：导出Markdown用于README或文档
  • 主题设计：为特定平台定制展示页面

离线网页加密生成器 v6.0

功能强大的离线网页加密工具，可以将HTML离线页面加密后套上一个密钥输入验证解锁的页面。支持密码、文件或远程API验证，纯前端实现，无需服务器支持。

AES加密 多种解锁方式 自定义模板 纯前端 离线可用

解锁方式：

  • 远程API验证（支持JSON/纯文本响应）
  • 本地密码解锁（支持多密码）
  • 文件锁解锁（使用文件哈希作为密钥）

主要功能模块

  • 数据源与密钥配置： 自动生成随机密钥，用于加密原始HTML内容
  • 解锁策略配置： 支持三种解锁方式（远程API、本地密码、文件锁）
  • 自定义模板编辑： 内置Monaco代码编辑器，支持语法高亮
  • 开发指南： 提供完整的JavaScript API，便于开发者自定义交互逻辑

适用场景：

  • 加密个人笔记或私密文档
  • 企业内部文件分发与权限控制
  • 付费内容预览与解锁
  • 基于文件哈希的版权保护
  • 离线可用的网页内容保护方案

bili-lx 直播间点歌bot

一个用于B站直播间的点歌机器人程序，支持通过弹幕和礼物触发点歌功能，具有完善的权限管理系统和实时数据显示功能。

已终止维护 弹幕控制 权限系统 OBS集成 礼物触发

主要功能：

  • 弹幕点歌控制： 观众通过发送特定弹幕点歌
  • 完善的权限系统： 白名单/黑名单用户管理
  • OBS实时显示： 当前播放歌曲信息显示
  • 礼物触发点歌： 特定礼物可触发点歌权限
  • 数据统计与历史记录： 点歌数据统计与查询

重要提醒：

1. 此项目现已终止更新与维护，漏洞等问题较严重，不建议使用

2. 在直播间播放版权音乐可能涉嫌侵权，请谨慎使用

3. 仅供学习交流使用，请勿用于商业用途

技术配置

  • Cookie配置： 支持获取完整用户名
  • 权限管理： 白名单/黑名单用户权限管理
  • 实时交互： 与B站直播间弹幕系统集成
  • 数据存储： 点歌历史记录与用户数据统计

项目状态：

  • 项目已归档，不再接受功能更新
